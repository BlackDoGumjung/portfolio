## ✨포트폴리오 페이지

https://blackdogumjung.github.io/portfolio/

### 🤔Summary
지금까지의 작업들을 모아둔 개인 포트폴리오 페이지입니다.

주요 기능
* 디자인, 코딩 100%, 반응형 
* flex를 이용한 work 페이지, CSS animation 활용
* 왼쪽 상단의 클릭 기능 이용한 페이지 
* EMAIL.JS를 이용하여 실제 메시지를 받을 수 있도록 구현 
* fontawesome 활용
